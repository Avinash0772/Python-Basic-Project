# Basic PRO - Python Projects Collection

A collection of Python projects showcasing fundamental programming concepts and game development.

## Projects

### 1. Rent Calculator
- **Description**: An application to calculate monthly rent and related expenses
- **File**: `RENT CALCULATOR/sourceCode.py`
- **How to Run**: 
  ```bash
  python "RENT CALCULATOR/sourceCode.py"
  ```
- **Purpose**: Automate rent and expense calculations for easy tracking

### 2. Rock Scissor & Paper Game
- **Description**: Classic Rock, Paper, Scissors game with interactive gameplay
- **File**: `ROCK SCISSOR & PAPER GAME/code.py`
- **How to Run**: 
  ```bash
  python "ROCK SCISSOR & PAPER GAME/code.py"
  ```
- **Purpose**: Play against the computer with randomized moves and win/loss tracking

### 3. Tic Tac Toe Game
- **Description**: Traditional Tic Tac Toe game on a 3x3 grid
- **File**: `TIC TAC TOE GAME/code.py`
- **How to Run**: 
  ```bash
  python "TIC TAC TOE GAME/code.py"
  ```
- **Purpose**: Interactive game with logic implementation and win condition detection

## Prerequisites

- Python 3.x installed on your system
- Basic knowledge of running Python scripts

## Installation

1. Clone or download this repository
2. Navigate to the project directory:
   ```bash
   cd "Basic PRO"
   ```
3. Run any project using the commands listed above
